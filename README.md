# Reply Flink Training

## Panoramica
Questo repository contiene esercizi pratici per Apache Flink.

## Prerequisiti
- Java 11+ e Maven
- Docker (per connettori opzionali/servizi demo)

## Struttura del Repository

Questo repository Ã¨ organizzato in due package principali:

- **`exercises`**: Contiene gli esercizi di training con commenti TODO e implementazioni incomplete
- **`solutions`**: Contiene le soluzioni complete a tutti gli esercizi

## Supporto Linguistico

- **branch `main`**: Tutti gli esercizi e la documentazione sono in **Inglese**
- **branch `italian`**: Gli stessi esercizi e documentazione tradotti in **Italiano**

Per passare alla versione inglese:
```bash
git checkout main
```

## Iniziare
1. Clonare il repository.
2. Eseguire `mvn clean package` per verificare le dipendenze.
3. Seguire le istruzioni degli esercizi in ogni modulo; completare il codice all'interno dei package degli esercizi forniti.
